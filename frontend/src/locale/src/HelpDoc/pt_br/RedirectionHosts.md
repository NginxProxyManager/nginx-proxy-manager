## O que é um Host de Redirecionamento?

Um Host de Redirecionamento irá redirecionar requisições do domínio de entrada e enviar o
visitante para outro domínio.

O motivo mais comum para usar este tipo de host é quando seu site muda
de domínio, mas você ainda tem links de mecanismos de busca ou referenciadores apontando para o domínio antigo.
